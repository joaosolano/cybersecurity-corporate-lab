## 🎯 Sobre o Projeto

O **CyberTécnica LTDA** reproduz o funcionamento de uma empresa real em um HD externo portátil, consolidando conhecimentos práticos em:

- Segurança da Informação
- Administração Linux/Windows
- Redes de Computadores
- Computação Forense
- Pentest e Hardening
- Virtualização e Monitoramento

---

## 🏛️ Arquitetura

Empresa fictícia organizada em três andares físicos, com estação dedicada a ataques. Cada ambiente simula setores e cenários reais de uma corporação.

---

## 🛠️ Tecnologias Utilizadas

| Categoria | Tecnologias |
|-----------|-------------|
| Virtualização | VirtualBox |
| SOs | Ubuntu Server, Windows 10, Kali Linux |
| Redes | Bridge, NAT, TCP/IP, SMB |
| Firewall | UFW, Windows Defender |
| Serviços | Apache2, PHP |
| Análise | Wireshark, Nmap |
| Criptografia | VeraCrypt (AES-256) |
| Forense | dcfldd, MD5, SHA-1 |
| Documentação | GitHub, Markdown, Python (Pillow) |

---

## 🔐 Destaques Técnicos

- Política UFW: `deny incoming / allow outgoing` (menor privilégio)
- Preservação forense com hashing criptográfico
- Criptografia de dados sensíveis
